El granito, es una combinación de minerales como el cuarzo, feldespato y mica. Es la piedra natural por excelencia para el uso que se le quiera dar.
  • Para cocinas: recomendado
  • Para baños: recomendado
  • Para revestimientos: recomendado
  • Para exteriores: recomendado
El Marmol es carbonato de calcio compactado naturalmente por miles de años de formacion rocosa. Es un producto fino y resistente, puede lucirse en diferentes ambientes de tu hogar y darle mucho estilo y clase.
  • Para cocinas: NO recomendado
  • Para baños: recomendado
  • Para revestimientos: recomendado
  • Para exteriores: NO recomendado
Los Materiales sintéticos (Silestone, NeoLith, Santa Margheritta) son una base cuarzo natural unificado a base de resinas Microban(TM), para la decoración de cocinas y baños a partir de un tratamiento que ofrece bellos y variados colores, texturas y unas altas prestaciones. Su uso es casi como el de granito, pero tiene menos durabilidad.
  • Para cocinas: recomendado
  • Para baños: recomendado
  • Para revestimientos: recomendado
  • Para exteriores: NO recomendado
